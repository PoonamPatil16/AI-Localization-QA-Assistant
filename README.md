# AI Localization QA Assistant

An AI-assisted localization quality assurance tool designed to automate translation validation and terminology consistency checks in multilingual datasets.

## 🚀 Features

- Detects missing translations
- Flags terminology inconsistencies
- Identifies untranslated source text
- Uses scalable terminology database
- Provides AI-based correction suggestions
- Supports CSV upload & QA report export

## 🧠 Problem It Solves

Manual translation QA is time-consuming and prone to inconsistency.

This tool automates quality checks by analyzing bilingual content and identifying:

- Missing entries
- Terminology violations
- Formatting issues

## ⚙️ Tech Stack

- Python
- Flask
- CSV Processing
- Rule-based AI Suggestions
